# 📊 E-Commerce Customer Behavior Analysis & Insights

## 📌 Overview

This project explores customer behavior in an e-commerce environment using a dataset of over 17,000 transactions.
The goal is to uncover patterns, trends, and actionable insights that can help businesses improve decision-making and customer experience.

---

## 🎯 Objectives

* Analyze customer purchasing behavior
* Identify trends over time (monthly, quarterly)
* Understand customer distribution across cities
* Detect patterns in orders and transactions
* Provide business insights for better strategy

---

## 📂 Dataset

* Source: E-commerce dataset (17,049 transactions)
* Includes:

  * Customer ID
  * Order ID
  * Purchase date
  * City
  * Transaction details

---

## 🔍 Key Analysis Steps

### 1. Data Understanding

* Dataset exploration (`head`, `info`, `describe`)
* Checking unique values and data types

### 2. Data Preparation

* Converted date column to datetime format
* Created new features:

  * Year
  * Month
  * Quarter
  * Day of Week

### 3. Exploratory Data Analysis (EDA)

* Total transactions, customers, and orders
* Customer distribution by location
* Time-based trends (monthly & yearly)
* Behavioral patterns in purchasing

### 4. Statistical Analysis

* Applied statistical methods (Chi-square test)
* Identified relationships between variables

---

## 📊 Key Insights

* Significant variation in customer activity across months
* Certain cities contribute more to total transactions
* Customer behavior shows seasonal patterns
* Data suggests opportunities for targeted marketing

## 📈 Business Recommendations
